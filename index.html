Browser JavaScript

From jQuery to Knockout to React, it's been a pretty active decade in browser-based JS.  I've used Grunt, Gulp, Browserify, and Webpack to package SPAs.  Karma and Jasmine are my testers of choice.  I place a high focus on building reusable and testable UI.

NodeJS

I've built RESTful APIs using Express, Sails, and Node-RED.  Waterline and Sequelize have been the ORM layer for MongoDB, SQL, and Postgres projects.  I've used Redis for caching, short-life heartbeats, and geospatial collision detection.  SOA-all-the-way (I've been known to use dependency injection) to maximize the test surface of business logic while ensuring scalability with microservices.

DevOps, IoT, UAVs, etc.

I utilize a AWS/Vagrant/Jenkins/New Relic setup for basic DevOps.  I'm familiar with 3D Robotic's Autopilot SDK to control their drone products (raw access via Serialport), CylonJS for rapid IoT prototyping, and Node-RED for more complex cases to enforce functional design.
